Welcome to what might be the world's most peculiar experiment in software development!

Justin Cappos had the core idea for this project—a technological progression game from primitive tools to modern marvels. Unlike conventional development, Justin embraced the unconventional approach of partnering with Large Language Models to bring this vision to life.

A substantial portion of the code, especially the web aspects you're currently experiencing, was nearly entirely written by OpenAI technologies and GitHub Copilot. The early backend was more human-crafted after some LLM-generated code struggles, while the frontend emerged almost fully from AI suggestions.

Perhaps most intriguingly, this project is now primarily maintained by GitHub's Copilot Agent—yes, an AI is updating this AI-written code. This experiment might represent either:

  1. A spectacular insight into the future of software maintenance, where AIs manage their own creations with minimal human oversight, or
  2. A colossal failure waiting to happen, like asking a toddler to fix a broken vase with superglue and hoping for the best.

Either way, this project serves as a fascinating demonstration of how far LLM capabilities extend into complex software development tasks—and where they still fall short. So whether we're witnessing the dawn of AI-maintained software or just an amusing academic experiment, you're looking at code that's been touched more by silicon intelligence than human hands.
